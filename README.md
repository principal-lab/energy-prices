# Energy Prices Widget

Real-time energy prices widget for Android (PWA).

## Features
- WTI and Brent Crude Oil prices
- JET-A1 Fuel prices
- ACT & NSW Electricity prices
- Premium Unleaded fuel prices (ACT)
- Prices displayed in USD and AUD
- Manual update button
- Works offline after first load

## Installation

### Deploy to GitHub Pages
1. Generate icons: Open `generate_icons.html` and download both icons
2. Create a new GitHub repository
3. Upload all files (except generate_icons.html)
4. Enable GitHub Pages in repository settings

### Install on Android
1. Visit your GitHub Pages URL on Chrome (Android)
2. Tap menu (⋮) → "Add to Home screen"
3. App appears on home screen

## Files
- `index.html` - Main application
- `manifest.json` - PWA configuration
- `service-worker.js` - Offline support
- `icon-192.png` - Small app icon (generate first)
- `icon-512.png` - Large app icon (generate first)

See `ENERGY_APPS_GITHUB_INSTRUCTIONS.md` for detailed setup instructions.
